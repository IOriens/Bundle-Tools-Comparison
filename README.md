## Intro
Testing all kinds of module bundlers like webpack, fuse-box and browserify.

## Usage

```
# Install deps 
cnpm i

# cd to import or require dir.
cd import

# Test webpack 2 
node webpack.js 

# Test webpack 1 
node webpack.1.js

# Test fuse-box
node fuse.js

# Test browserify
node browserify.js
```
Checkout built files in `dist` directory.
